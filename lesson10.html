Lesson 10: Functions

Organize code into reusable blocks with functions.

Introduction

Functions are reusable blocks of code that perform specific tasks.

They help organize code and avoid repetition.

Examples

Example 1: Simple Function

Python 
# Defining a function
def greet(name):
    print(f"Hello, {name}!")

# Calling the function
greet("Alice")
greet("Bob")

Example 2: Function with Return Value

Python 
# Function with parameters and return value
def add_numbers(a, b):
    return a + b

result = add_numbers(5, 3)
print("5 + 3 =", result)

Example 3: Function with Default Parameters

Python 
# Function with default parameters
def greet_with_title(name, title="Mr."):
    return f"Hello, {title} {name}!"

print(greet_with_title("Smith"))
print(greet_with_title("Johnson", "Dr."))

Example 4: Function with Multiple Return Values

Python 
# Function with multiple return values
def get_name_and_age():
    return "Alice", 25

name, age = get_name_and_age()
print("Name:", name, "Age:", age)

Key Points

  • def: Keyword to define a function
  • Parameters: Values passed to the function
  • return: Send value back from function
  • Default parameters: Values used if none provided
  • Functions can return multiple values as tuples
