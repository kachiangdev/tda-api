Lesson 6: Conditions and If-Else

Make decisions in code.

Introduction

if, elif, else let you branch your logic.

Examples

Example 1: Simple If Statement

age = 20
if age >= 18:
    print("Adult")

This will output: Adult

Example 2: If-Else Statement

num = -3
if num > 0:
    print("Positive")
else:
    print("Zero or Negative")

This will output: Zero or Negative

Example 3: If-Elif-Else Statement

score = 85
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("C or lower")

This will output: B

Comparison Operators

  • == Equal to
  • != Not equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to
