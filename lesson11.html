Lesson 11: Dictionaries

Store key-value pairs.

Introduction

Dictionaries map keys to values.

Examples

Example 1: Creating a Dictionary

person = {"name": "Alice", "age": 25}
print(person)

This will output: {'name': 'Alice', 'age': 25}

Example 2: Accessing Dictionary Values

person = {"name": "Alice", "age": 25}
print(person["name"])
print(person["age"])

This will output:

Alice
25

Example 3: Adding New Key-Value Pairs

person = {"name": "Alice", "age": 25}
person["city"] = "New York"
print(person)

This will output: {'name': 'Alice', 'age': 25, 'city': 'New York'}

Dictionary Operations

  • dict[key] - Access value by key
  • dict[key] = value - Add or update key-value pair
  • dict.get(key) - Safely access value (returns None if key doesn't exist)
  • dict.keys() - Get all keys
  • dict.values() - Get all values
  • dict.items() - Get all key-value pairs
