Lesson 13: Text Adventure Game

Build an interactive story.

Project Description

  • Print scenario text
  • Ask for user choices
  • Branch the story with if/else
  • Use variables to track progress

Example Concepts to Use

  • input()
  • if-else
  • Strings
  • Loops (optional)

How It Works

  1. We create a function to organize the game
  2. We print story text to set the scene
  3. We ask for user input to make choices
  4. We use if/else statements to branch the story
  5. Each choice leads to different outcomes
  6. We track the player's progress through variables
