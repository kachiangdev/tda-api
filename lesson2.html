Variables are containers that store data values. Think of them as labeled boxes where you can put different types of information.

  • Must start with a letter or underscore
  • Can contain letters, numbers, and underscores
  • Case-sensitive (name â‰  Name)
  • Cannot use Python keywords

ğŸ”¤ Creating Variables

In Python, you create variables by assigning values to them using the = operator:

Python 
# Creating variables
name = "Alice"
age = 25
height = 1.75
is_student = True

# Printing variables
print("Name:", name)
print("Age:", age)
print("Height:", height, "meters")
print("Is student:", is_student)

ğŸ” Checking Data Types

You can check the type of a variable using the type() function:

Python 
# Different data types
x = 10
y = 3.14
text = "Hello"
flag = False

# Checking types
print("Type of x:", type(x))
print("Type of y:", type(y))
print("Type of text:", type(text))
print("Type of flag:", type(flag))

Output:

<class 'int'>
<class 'float'>
<class 'str'>
<class 'bool'>

You can convert between different data types using built-in functions:

number = 123
string_number = str(number)  # Convert integer to string
print("String:", string_number, type(string_number))
