Lesson 5: Input from User

Read user input.

Introduction

input() always returns a string. Convert if needed.

Examples

Example 1: Basic Input

name = input("Enter your name: ")
print("Hi,", name)

This will output: Hi, [user's name]

Example 2: Converting to Integer

age = int(input("Enter your age: "))
print("Next year you'll be", age + 1)

This will output: Next year you'll be [age + 1]

Example 3: Mathematical Operations

num = int(input("Number: "))
print(num * 2)

This will output: [number * 2]

Key Points

  • input() always returns a string
  • Use int() to convert to integer
  • Use float() to convert to decimal
  • The prompt text is optional but helpful
  • Always handle potential conversion errors
