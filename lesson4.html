Lesson 4: String Manipulation

Combine and modify text.

Introduction

Strings can be joined, changed case, and formatted.

Examples

Example 1: String Concatenation

first = "Alice"
last = "Smith"
print(first + " " + last)

This will output: Alice Smith

Example 2: String Methods

text = "hello world"
print(text.upper())
print(text.capitalize())

This will output:

HELLO WORLD
Hello world

Example 3: F-Strings (String Formatting)

name = "Alice"
age = 25
print(f"{name} is {age} years old")

This will output: Alice is 25 years old

Common String Methods

  • .upper() - Convert to uppercase
  • .lower() - Convert to lowercase
  • .capitalize() - Capitalize first letter
  • .title() - Capitalize each word
  • .strip() - Remove whitespace from ends
  • .replace(old, new) - Replace text
