Lesson 9: Loops

Repeat tasks.

Introduction

for loops iterate over sequences.

Examples

Example 1: Range Loop

for i in range(5):
    print(i)

This will output:

0
1
2
3
4

Example 2: List Loop

for fruit in ["apple", "banana"]:
    print(fruit)

This will output:

apple
banana

Example 3: Sum with Loop

numbers = [1, 2, 3]
total = 0
for num in numbers:
    total += num
print(total)

This will output: 6

Loop Types

  • for item in list: - Loop through list items
  • for i in range(n): - Loop n times (0 to n-1)
  • for i in range(start, end): - Loop from start to end-1
  • while condition: - Loop while condition is True
