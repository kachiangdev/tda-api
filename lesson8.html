Lesson 8: Lists

Store multiple values.

Introduction

Lists are ordered, mutable collections.

Examples

Example 1: Creating and Printing Lists

fruits = ["apple", "banana", "cherry"]
print(fruits)

This will output: ['apple', 'banana', 'cherry']

Example 2: Accessing List Elements

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # First element
print(fruits[1])  # Second element

This will output:

apple
banana

Example 3: Adding Elements

fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits)

This will output: ['apple', 'banana', 'cherry', 'orange']

Common List Operations

  • list.append(item) - Add item to end
  • list.remove(item) - Remove specific item
  • list.pop() - Remove and return last item
  • len(list) - Get list length
  • list[index] - Access element by index
  • list.sort() - Sort list in place
