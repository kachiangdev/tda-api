Lesson 7: Number Guessing Game

Build your first game.

Project Description

  • Computer picks random number 1‚Äì20
  • User keeps guessing until correct
  • Program says "Too high", "Too low", or "Correct!"
  • Track and print the number of guesses

Example Concepts to Use

  • import random
  • while loop
  • if-else branching
  • User input

How It Works

  1. random.randint(1, 20) generates a random number
  2. while True: creates an infinite loop
  3. We get user input and convert it to integer
  4. We compare the guess to the secret number
  5. If correct, we print success message and break
  6. Otherwise, we give a hint and continue
