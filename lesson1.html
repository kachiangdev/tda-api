Lesson 1: Introduction to Python

Run Python code online and print output.

üéØ What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It's perfect for beginners and is used in web development, data science, artificial intelligence, and more.

üåü Why Python?

  • Easy to read and write
  • Large community and extensive libraries
  • Great for beginners
  • Versatile and powerful

üìù Your First Python Program

Let's start with the traditional "Hello, World!" program:

Python 
print("Hello, World!")

This simple line of code will output: Hello, World!

üî§ The Print Function

The print() function is used to display text or values on the screen. Here are some examples:

Python 
# Print a simple message
print("Hello, World!")

# Print multiple items
print("Welcome", "to", "Python!")

# Print with newlines
print("Line 1\nLine 2")

Output:

Hello, World!
Welcome to Python!
Line 1
Line 2

üí° Key Concepts

üìù Comments

Use # to add comments that explain your code:

# This is a comment
print("This is code")

üî§ Strings

Text in Python is called a string and must be enclosed in quotes:

print("This is a string")
print('This is also a string')

üìã Functions

Functions are reusable blocks of code. print() is a built-in function:

print("Hello")  # Function call
